# Meet Captions Translator

Chrome-расширение для перехвата субтитров Google Meet и автоперевода на русский язык через OpenRouter API.

## Возможности

- Перехват live-субтитров Google Meet через MutationObserver
- Стриминг оригинального текста в оверлее поверх видео (обновляется на месте, без дублирования строк)
- Автоперевод на русский через OpenRouter API (любая модель)
- Умное определение конца фразы: пунктуация (800мс), пауза в речи (3 сек), макс. накопление (12 сек)
- Перевод отображается зелёным текстом под оригиналом в оверлее и дублируется в консоль

## Установка

1. Откройте `chrome://extensions`
2. Включите **Режим разработчика** (переключатель в правом верхнем углу)
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку `translate`

## Настройка

1. Нажмите на иконку расширения в Chrome
2. Введите OpenRouter API ключ (`sk-or-v1-...`)
3. Выберите модель (по умолчанию Gemini 2.5 Flash Lite)
4. Нажмите **Сохранить**

## Использование

1. Откройте Google Meet и подключитесь к звонку
2. Включите субтитры (кнопка **CC** внизу экрана)
3. Оригинальный текст появится в оверлее внизу экрана
4. Перевод появится автоматически после завершения фразы

### Консоль (F12)

```
[CC] Speaker: original text
[>>] Speaker: перевод
```

## Структура

```
manifest.json   — конфигурация расширения (Manifest V3)
content.js      — перехват субтитров, оверлей, логика таймингов
background.js   — service worker, запросы к OpenRouter API (SSE стриминг)
popup.html/js   — popup настроек (API ключ, выбор модели)
```
